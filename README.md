# XSL Quest - Генератор тестов из Excel

Веб-приложение для создания интерактивных тестов из Excel файлов.

## Возможности

- Загрузка тестов из Excel файлов (.xlsx, .xls)
- Автоматическая генерация уникальных ссылок для тестов
- Поддержка множественного выбора ответов
- Сохранение результатов тестирования
- Доступ к тестам по сети

## Требования

- Node.js 18+
- npm или yarn

## Быстрый старт

### Установка зависимостей

```bash
npm run install-backend
```

### Запуск сервера

```bash
npm start
```

Сервер запустится на http://localhost:5000

## Формат Excel файла

Excel файл должен содержать следующие колонки:

| Вопрос | Вариант 1 | Вариант 2 | Вариант 3 | Вариант 4 | Правильный ответ |
|--------|-----------|-----------|-----------|-----------|------------------|
| Текст  | Вариант   | Вариант   | Вариант   | Вариант   | 1,2 или A,B      |

## API

### POST /api/upload
Загрузка Excel файла и создание теста

### GET /api/test/:testId
Получение данных теста

### POST /api/submit/:testId
Отправка ответов на тест

## Технологии

**Backend:**
- Node.js + Express
- ExcelJS для парсинга
- Multer для загрузки файлов
- CORS для кросс-доменных запросов

**Frontend:**
- Vanilla JavaScript
- HTML5 + CSS3
- Адаптивный дизайн
